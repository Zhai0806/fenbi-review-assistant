# 用户能力画像

## 备考基本情况

- **备考周期**：1年（2025年7月至今），非短期冲刺
- **实战经历**：6次公考/事考实战
- **行测/职测实战成绩**：稳定在60分以上（事考职测折算后）
- **备考状态**：2026年7月起转为在职备考，工作日备考时间受限
- **策略定位**：系统长期提升，不接受「放弃某模块」的策略建议（除数量关系可战略性取舍）
- **目标**：行测70+，三大块平均正确率80%+

## 做题顺序

言语理解与表达(1) → 资料分析(2) → 判断推理(3) → 政治理论(4) → 常识判断(5) → 数量关系(6)

## 模块时间预算

| 模块 | 时间预算 | 备注 |
|------|---------|------|
| 言语理解与表达 | 20-25 min | — |
| 资料分析 | 25-30 min | 针对4组题（20题） |
| 判断推理 | 30-35 min | — |
| 政治理论+常识判断+数量关系 | 30-40 min | 三小块共享 |

## 各模块详细情况

### 言语理解与表达（优势模块，目标正确率80%+）

- **逻辑填空**：已知弱点。能根据空前空后及句子逻辑关系推断，但成语/实词储备不足，导致不了解选项词语的实际含义而选错。
- **阅读理解**：相对稳定。

### 资料分析（优势模块但有瓶颈，目标正确率80%+）

- **算法选择**：考场上倾向于「厂除」（直除），虽学过百化分、假设分配等技巧，但在紧张环境下会选"最不麻烦"的算法，导致耗时偏长。
- **时间与正确率匹配**：频繁出现用时偏长但正确率未相应提升的现象。
- **提升方向**：在允许范围内丰富算法选择，大量做题培养熟练度，使高级算法在考场也能自然调用。

### 判断推理（优势模块，目标正确率80%+）

- **图形推理**：取决于短时间内能否察觉规律。若不能，先「跳岛」等三大块结束后回头。回头仍无结果则排除后选最接近的。
- **逻辑判断-翻译题**：公式记忆不清，掌握不牢固。
- **逻辑判断-组合排列题**：不知从何下手。
- **类比推理**：相对稳定。

### 政治理论（薄弱模块，需补基础）

- 知识储备不足。有时可根据题目选项逻辑排除部分错误选项获得高正确率，有时很低，波动大。
- **提升方向**：需要频繁记忆，系统性积累。

### 常识判断（薄弱模块，波动大）

- 依赖日常积累，正确率波动极大。
- **提升方向**：长期积累，不追求短期突破。

### 数量关系（弱项，战略性取舍）

- **策略**：时间充裕则做5题蒙5题；不充裕则全蒙同一选项。
- **提升方向**：需要系统性学习，但不能挤压三大块时间。

## 关键非知识因素

- **「状态」**：考试状态和平日备考状态是影响成绩的关键变量。考场紧张会降级算法选择（如只能用厂除而调不出百化分）。
- **备考节奏**：在职备考（2026年7月起），需要高效利用有限时间。

## 已识别的具体错因模式

- 逻辑填空：选项词语不认识 → 逻辑推断正确但选不出 → 蒙
- 逻辑判断翻译题：公式记忆混乱 → 推不出 → 蒙
- 组合排列：不知道从哪里入手 → 放弃
- 图推：短时间看不出规律 → 跳过后回头 → 仍看不出 → 排除后蒙
- 资料分析：考场紧张 → 只能用厂除 → 耗时偏长 → 挤压后续模块
